# 9jaCash

9jaCash is a premium high-end online casino lucky wheel application with a fully integrated secure Node.js/SQL backend, Paystack payments, verification flows, and responsive administrative controls.

## Features
- **AAA Casino Lucky Wheel:** Realistic coin drop, spin physics, background music, audio cues, and premium wallet collection animation.
- **Node.js/SQL Backend:** Built using Express, PostgreSQL, and SQLite with automated schema verification and data purging.
- **Paystack Payment & Verification:** Automated deposit validations, proof uploads, and account activation refund workflows.
- **Admin & Junior Admin Panels:** Fully mobile-responsive consoles to manage Referred networks, verify uploads, and check payment receipts.
